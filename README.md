# Match3_Stone_Solor_System

어린 왕자
행성의 돌
애정 주기

어린 왕자는 무언가를 잃어버렸습니다. 
스스로가 무엇을 잃어버렸는지 기억을 하지 못합니다.
우주를 찾아 소중한 것을 찾아 떠납니다.
행성을 여행하며 별의 조각들을 모아 행성을 돌아봅니다.
지구에서부터 먼 은하의 별까지 각 별의 조각을 모으는 여행을 떠납니다.

하트 - 시간이 지나면 채워짐

행성 컬랙션 - 30/60/90/100


1단계 지구 -> 달 -> 화성 ->

2단계 수성 -> 금성 -> 목성 ->

3단계 토성 -> 천왕성 -> 태양 ->

지구
